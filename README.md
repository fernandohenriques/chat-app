# Chat App

## Getting Started

Front-end web project of an open chat, written with ReactJS + Redux + React DOM Router.

### Pre requisites

```
$ node --version
v10.9.0

$ Yarn --version
1.9.0

```


### How to Run

```
git clone https://github.com/fernandohenriques/chat-app.git
cd chat-app/
yarn install
yarn start
```


### Running tests

#### Linter validations
```
cd chat-app/
yarn lint
```
